### Usage
```
$ docker pull chenhw2/frp

$ docker run -d \
    -e DOMAIN=%I
    -v /etc/ssl/caddy:/etc/ssl/caddy
    -v /var/gitea-data:/data
    -p 80:80 -p 443:443
    chenhw2/gitea:latest
```
